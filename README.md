## Built with
- macOS Monterey 12.4 (M1)
- Python >= 3.9

- `brew install cmake libomp`
- `pip install pandas numpy seaborn matplotlib scipy scikit-learn xgboost graphviz`
- `conda install lightgbm -y`
- `brew install graphviz`
